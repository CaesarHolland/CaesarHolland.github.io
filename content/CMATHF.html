C++ Math Function

04 / 26 / 2020

三角函数 trigonometric function

  1. double sin(double)
  2. double cos(double)
  3. double tan(double)

反三角函数 Anti-trigonometric function

  1. double asin(double); //结果介于[-Pi/2 , Pi/2]
  2. double acos(double); //结果介于[0 , Pi]
  3. double atan(double); //反正切（主值）结果介于[-Pi/2 , Pi/2]
  4. double atan2(double); //反正切（整圆值）结果介于[-Pi/2 , Pi/2]

双曲三角函数 Hyperbolic trigonometric function

  1. double sinh(double);
  2. double cosh(double);
  3. double tanh(double);

指数与对数 Exponents and Logarithms

  1. double exp(double);
  2. double pow(double , double);
  3. double log(double); //e为底
  4. double log10(double); //其他对数可以用换底公式解决：log2(N)=log10(N)/log10(2)

取整函数 Rounding Function

  1. double ceil(double); //up rounding
  2. double floor(double); //down rounding

绝对值 Absolute value function

double fabs(double);

标准化浮点数 Standardized floating point

  1. double frexp(double f , int *p); //f=x*2^p , 已知f、p求x
  2. double ldexp(double x , int p); //已知x、p求f

取整与取余 Rounding and surplus

  1. double modf(double , double*); //将参数的整数部分通过指针回传，返回小数部分
  2. double fmod(double , double); //返回两参数相除的余数

平方根

double sqrt(double);
